## operator GE

**_expression1_ GE _expression2_**

Returns true (-1) if _expression1_ is greater than or equal to _expression2_, or returns false (0) if it is not.

_* Non ML compatible usage_
```
    a = 10.0
    while a ge 5.0
        a = a - 1.0
        endm
```
