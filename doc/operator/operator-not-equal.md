## operator !=

**_expression1_ != _expression2_**

Is not equal to.

Used only within [.IF](../directive/dot-if.md), [.WHILE](../directive/dot-while.md), or [.REPEAT](../directive/dot-repeat.md) blocks and evaluated at run time, not at assembly time.
