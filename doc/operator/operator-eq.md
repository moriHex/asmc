## operator EQ

**_expression1_ EQ _expression2_**

Returns true (-1) if _expression1_ equals _expression2_, or returns false (0) if it does not.

_* Non ML compatible usage_
```
    a = 1.0
    while a eq 1.0
        a = a - 1.0
        endm
```
