# Return Code

Sets size from a function call to _xx_.

**.IF**_xx_, **.WHILE**_xx_, **.UNTIL**_xx_, and **.ASSERT**_x_ <_expression_>.

- **B** BYTE
- **W** WORD
- **D** DWORD
- **SB** SBYTE
- **SW** SWORD
- **SD** SDWORD
